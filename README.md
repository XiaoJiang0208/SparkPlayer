## SparkPlayer
**SparkPlayer** 是一款轻量、高效的Linux AI音视频播放器，基于 FFmpeg 开发，支持多种媒体格式，提供流畅的播放体验和灵活的播放控制。
**特色功能**
- AI动画角色识别与面部表情识别
- 音频风格识别
